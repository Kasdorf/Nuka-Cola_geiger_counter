# Nuka-Cola_geiger_counter
Arduino sketch that uses a distance sensor and piezo buzzer to mimick a geiger counter sound, that clicks faster, the closer you get.
Pins:
  Distance Sensor:
     - VCC ➡ 5V
     - Trig ➡ 11
     - Echo ➡ 12
     - GND ➡ GND

  Piezo Buzzer:
    - Short (neg) ➡ GND
    - Long (pos) ➡ 9

  NeoPixel Ring:
    - Data Input ➡ 0
    - Power ➡ 5V
    - GND ➡ GND
